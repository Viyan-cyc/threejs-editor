# 对话记录（Conversations）

本目录存放本项目人可读的**对话日志**，用于沉淀每次与 AI 协作的过程与决策。

## 记录约定

- **一个文件 = 一次会话**：文件名格式 `YYYY-MM-DD-主题.md`（如 `2026-06-15-项目初始化与骨架搭建.md`）。
- **按轮次组织**：每个 `## 轮次 N · 主题` 下记录
  - **用户**：用户消息（原文引用）。
  - **助手**：做了什么、改了哪些文件、关键决策。
  - **结果**：该轮的产出 / 验证结论。
- **顶部概要**：用 3–5 条点出本次会话整体做了什么。
- 文件内的相对链接用 `../../` 指向 `src/` 等代码位置，方便跳转。

## 谁来维护

由 AI（Claude）在每次会话中维护。机制：
- 每次新会话开始时，AI 读取本目录，**新建当天文件**（或追加到当天文件）。
- 每完成一个有意义的轮次后，**追加对应轮次记录**。
- 该约定已写入 Claude 的项目记忆，后续会话会自动延续。

> 如需临时关闭/调整记录粒度，直接告诉 AI 即可。
